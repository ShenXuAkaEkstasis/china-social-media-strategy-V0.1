# China Social Media Strategy Skill （V0.1)

## Overview

This Skill helps overseas companies develop a social media strategy before entering the China market.

It analyzes the business, recommends suitable Chinese social media platforms, suggests appropriate content formats, and generates an initial China GTM social media strategy.

This Skill focuses on strategy rather than execution.
